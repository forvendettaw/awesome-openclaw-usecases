# 微博热搜监控

> 实时追踪微博热搜榜单

## 定位

监控微博热搜榜，自动追踪指定话题热度变化。

## 触发条件

用户提到：
- 微博热搜
- 微博榜单
- 热搜监控
- 话题热度
- 热点追踪

## 功能

### 1. 热搜榜单

```python
# 获取实时热搜榜
df = ak.weibo_ranking()
```

返回字段：
- 排名
- 关键词
- 热度值
- 排名变化

### 2. 话题监控

监控指定关键词的热度变化：

```python
# 监控话题
topics = ["A股", "AI", "英伟达"]
for topic in topics:
    # 获取话题热度
    pass
```

### 3. 趋势分析

- 热度变化趋势
- 上榜时间
- 排名变化

```
用户: 今天的微博热搜有哪些
```
