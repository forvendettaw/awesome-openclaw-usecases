# 知乎问答追踪

> 追踪知乎特定问题和回答

## 定位

追踪知乎上的特定问题和回答，获取最新动态。

## 触发条件

用户提到：
- 知乎
- 知乎问答
- 追踪问题
- 知乎热榜

## 功能

### 1. 热榜获取

```python
# 获取知乎热榜
df = ak.zhihu_hot()
```

返回：
- 问题标题
- 热度值
- 回答数

### 2. 问题追踪

```python
# 追踪特定问题
question_id = "1234567890"
# 获取问题详情和回答
```

### 3. 回答监控

- 新增回答
- 高赞回答
- 回答趋势

## 安装

```bash
pip install akshare
```

## 使用示例

```
用户: 知乎今天热榜是什么
→ 获取知乎热榜 → 返回热榜问题

用户: 追踪"AI发展"相关问题
→ 搜索相关问题 → 监控新回答

用户: 这个知乎问题有哪些高赞回答
→ 获取问题详情 → 返回回答列表及赞同数
```
